# ✈️ Flight Route Dashboard

This is a simple real-time flight route dashboard built using:

- 🐍 Python
- 🌐 Streamlit
- 📊 Plotly
- 🧮 Pandas
- ✈️ AviationStack API

It displays:
- A preview of live airline flight data
- Filter by airline
- Top 10 routes (pie chart)
- Peak flight hours (bar chart)
